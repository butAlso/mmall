# happymmall （一期）网上购物商城 ==》（二期）将更新为分布式系统架构
happymmall是一个java spring web电商项目，分为用户模块、分类模块、商品模块、购物车模块、收货地址模块、支付模块、订单模块
七个模块组成，采用vsftpd做文件服务器，nginx做路由，可扩展性强。是spring实战进阶不错的一个项目。
